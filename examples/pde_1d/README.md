# 1D PDEs

One-space-dimension PDEs: heat and advection. Same style as [wave_2d](../wave_2d/). Julia: MethodOfLines.jl.

| File | What it does | Run |
|------|--------------|-----|
| [heat_1d.ein](heat_1d.ein) | 1D heat, Dirichlet BC | `python3 -m einlang examples/pde_1d/heat_1d.ein`; `python3 examples/pde_1d/run_heat_1d.py` for HTML |
| [advection_1d.ein](advection_1d.ein) | 1D advection, upwind, periodic | `python3 -m einlang examples/pde_1d/advection_1d.ein` |
